# ElixirBotForTelegram

Bot no Telegram para validacao de CPF, desenvolvido em Elixir como trabalho da disciplina de Linguagens de Programacao.

## Setup

### macOS / Linux

```bash
./setup.sh
```

### Windows (executar como Administrador)

```cmd
setup.bat
```
